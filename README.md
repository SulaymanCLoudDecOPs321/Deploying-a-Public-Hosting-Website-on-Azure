# Deploying-a-Public-Hosting-Website-on-Azure

🎯 Key Steps

Create a Resource Group

Create a Storage Account

Select:

Performance: Standard

Redundancy: LRS (for basic setup)

Enable Static Website Hosting under:

Data Management → Static Website

Set:

Index document: index.html

Error document (optional): 404.html

Upload website files to the $web container

Access the site using the provided Primary Endpoint URL
